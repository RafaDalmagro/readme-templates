## Instalação

### Pré-requisitos

- **Python**: versão **3.11**
- **pip** instalado
- **npm** instalado
- **node**: versão => 18

### Passo a passo instalação

1. **Instalar dependencias**: 
```bash
npm install
```
2. **Criando containers**: 
```bash
docker-compose up -d
```
3. **Variáveis de ambiente**: 
Copiar `.env.example` para o `.env.local`
